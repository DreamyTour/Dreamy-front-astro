# Políticas de Proteção de Dados Pessoais

No Turismo Perú Bolivia, respeitamos sua privacidade e protegemos seus dados pessoais.

## Coleta de Dados

Coletamos apenas as informações necessárias para prestar nossos serviços:

- Dados de contato (nome, email, telefone)
- Informações de passaporte para reservas de tours
- Preferências de viagem
- Dados de pagamento (de forma segura)

## Uso dos Seus Dados

Utilizamos seus dados para:

- Processar suas reservas e solicitações
- Comunicar com você sobre suas viagens
- Melhorar nossos serviços
- Cumprir obrigações legais

## Seus Direitos

Você tem direito a:

- Acessar seus dados pessoais
- Solicitar correção de dados incorretos
- Solicitar exclusão dos seus dados
- Opor-se ao tratamento dos seus dados

## Contato

Para exercer seus direitos, contacte: privacidad@turismo-peru-bolivia.com
